# Welcome to Node-Eshop!

A node Js Eshop, for educational purposes


## Build Process
```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm start
```

## MYSQL Database
```bash
  A .env file needs to be created with the variables
  DB_DATABASE,
	DB_USERNAME,
	DB_PASSWORD,
  for the connection with DB.
```
